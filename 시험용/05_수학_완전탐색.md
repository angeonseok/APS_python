## 4131 - 줄 세우기

### 1. 핵심 아이디어
- N명의 학생을 키 순서대로 줄 세울 때 이미 서있는 학생들의 위치 정보 활용
- 각 학생이 자기 앞에 몇 명이 있는지 정보 주어짐
- 뒤에서부터 삽입하는 방식으로 정렬
- 키 내림차순으로 정렬 후 "앞에 몇 명" 위치에 순서대로 삽입

### 2. 파이썬 코드

```python
import sys
input = sys.stdin.readline

T = int(input())
for tc in range(1, T + 1):
    N = int(input())
    students = []
    for _ in range(N):
        h, k = map(int, input().split())  # 키, 앞에 있는 사람 수
        students.append((h, k))

    # 키 내림차순, 키 같으면 k 오름차순
    students.sort(key=lambda x: (-x[0], x[1]))

    result = []
    for h, k in students:
        result.insert(k, h)  # k번째 위치에 삽입

    print(f'#{tc} ' + ' '.join(map(str, result)))
```

## 4123 - 시작

### 1. 핵심 아이디어
- N개의 수 중 두 수를 선택해 XOR 값이 최대가 되는 쌍 찾기
- 완전탐색으로 모든 쌍 시도 (N ≤ 1000이면 O(N²) 가능)
- XOR의 특성: 다른 비트가 많을수록 값이 큼

### 2. 파이썬 코드

```python
import sys
input = sys.stdin.readline

T = int(input())
for tc in range(1, T + 1):
    N = int(input())
    nums = list(map(int, input().split()))

    ans = 0
    for i in range(N):
        for j in range(i + 1, N):
            ans = max(ans, nums[i] ^ nums[j])

    print(f'#{tc} {ans}')
```

## 5658 - 보물상자 비밀번호

### 1. 핵심 아이디어
- N자리 16진수 문자열이 정사각형 4면에 배치
- 4번 회전하며 각 면의 N/4자리 16진수 값을 모두 수집
- 중복 없이 내림차순 정렬 후 K번째 값 출력
- 문자열 슬라이싱 + set으로 중복 제거

### 2. 파이썬 코드

```python
import sys
input = sys.stdin.readline

T = int(input())
for tc in range(1, T + 1):
    N, K = map(int, input().split())
    s = input().strip()
    L = N // 4  # 한 면의 길이

    passwords = set()
    for rot in range(N):  # N번 회전
        # 4면 추출
        for i in range(4):
            start = (rot + i * L) % N
            # 연속 슬라이싱 (순환)
            if start + L <= N:
                seg = s[start:start + L]
            else:
                seg = s[start:] + s[:start + L - N]
            passwords.add(int(seg, 16))  # 16진수 → 10진수

    sorted_pw = sorted(passwords, reverse=True)
    print(f'#{tc} {sorted_pw[K-1]}')
```
