## 2383 - 점심 식사시간

### 1. 핵심 아이디어
- N×N 지도에서 사람들이 계단으로 이동 후 내려가는 최소 시간
- 계단은 2개, 각 사람을 두 계단 중 어디에 배정할지 부분집합으로 결정
- 배정 후 각 계단별로 도착 시간 정렬 → 동시에 최대 3명만 이용 가능
- 시뮬레이션으로 모든 사람이 내려가는 시간 계산

### 2. 파이썬 코드

```python
import sys
input = sys.stdin.readline

def calc_time(people_times, stair_len):
    # people_times: 각 사람이 계단 입구에 도착하는 시각
    times = sorted(people_times)
    end_times = []  # 각 사람이 계단 완료 시각
    for t in times:
        # 현재 계단 사용 중인 사람 수 (3명 제한)
        in_use = [e for e in end_times if e > t]
        if len(in_use) >= 3:
            # 가장 빨리 끝나는 사람 기다림
            wait = sorted(in_use)[0]
            t = wait
        end_times.append(t + stair_len)
    return max(end_times) if end_times else 0

T = int(input())
for tc in range(1, T + 1):
    N = int(input())
    board = [list(map(int, input().split())) for _ in range(N)]

    people = []
    stairs = []
    for i in range(N):
        for j in range(N):
            if board[i][j] == 1:
                people.append((i, j))
            elif board[i][j] == 2:
                stairs.append((i, j))

    P = len(people)
    ans = float('inf')
    s0, s1 = stairs[0], stairs[1]
    s0_len = board[s0[0]][s0[1]]
    s1_len = board[s1[0]][s1[1]]

    # 각 사람을 계단 0 또는 계단 1에 배정 (부분집합)
    for mask in range(1 << P):
        g0, g1 = [], []
        for i in range(P):
            px, py = people[i]
            if mask & (1 << i):
                g0.append(abs(px - s0[0]) + abs(py - s0[1]) + 1)
            else:
                g1.append(abs(px - s1[0]) + abs(py - s1[1]) + 1)
        t0 = calc_time(g0, s0_len)
        t1 = calc_time(g1, s1_len)
        ans = min(ans, max(t0, t1))

    print(f'#{tc} {ans}')
```

## 2477 - 차량 정비소

### 1. 핵심 아이디어
- 접수 창구 A개, 정비 창구 B개로 나눠진 정비소 시뮬레이션
- 매 시간 단위로 고객 도착 → 접수 → 정비 과정 처리
- 우선순위: 빈 창구 중 번호 낮은 것 배정
- A번 접수 창구 + B번 정비 창구를 거친 고객 수 출력

### 2. 파이썬 코드

```python
import sys
input = sys.stdin.readline

T = int(input())
for tc in range(1, T + 1):
    A, B, Q = map(int, input().split())
    a_times = list(map(int, input().split()))  # 접수 창구 처리 시간
    b_times = list(map(int, input().split()))  # 정비 창구 처리 시간
    target_a, target_b = map(int, input().split())
    customers = list(map(int, input().split()))  # 각 고객 도착 시간

    # 창구 상태: 언제까지 사용 중인지
    a_end = [0] * A  # 접수 창구 종료 시각
    b_end = [0] * B  # 정비 창구 종료 시각
    a_window = {}    # 고객 -> 접수 창구 번호
    b_window = {}    # 고객 -> 정비 창구 번호

    wait_b = []  # 접수 완료 후 정비 대기 (시각, 고객번호)

    # 이벤트 처리 시뮬레이션
    customers_with_id = [(t, i+1) for i, t in enumerate(customers)]
    customers_with_id.sort()

    # 접수 창구 배정
    a_queue_info = []  # (접수완료시각, 고객, 창구번호)
    for arrival, cid in customers_with_id:
        # 가장 빨리 비는 창구 (번호 낮은 것 우선)
        chosen = -1
        chosen_end = float('inf')
        for i in range(A):
            end = a_end[i]
            avail = max(end, arrival)
            if avail < chosen_end or (avail == chosen_end and i < chosen):
                chosen_end = avail
                chosen = i
        start = max(a_end[chosen], arrival)
        finish = start + a_times[chosen]
        a_end[chosen] = finish
        a_window[cid] = chosen + 1  # 1-indexed
        a_queue_info.append((finish, cid))

    # 정비 창구 배정 (접수 완료 시각 순 + 번호 순)
    a_queue_info.sort()
    for finish_a, cid in a_queue_info:
        chosen = -1
        chosen_end = float('inf')
        for i in range(B):
            avail = max(b_end[i], finish_a)
            if avail < chosen_end or (avail == chosen_end and i < chosen):
                chosen_end = avail
                chosen = i
        b_end[chosen] = max(b_end[chosen], finish_a) + b_times[chosen]
        b_window[cid] = chosen + 1  # 1-indexed

    ans = sum(1 for cid in range(1, Q+1)
              if a_window.get(cid) == target_a and b_window.get(cid) == target_b)

    print(f'#{tc} {ans}')
```

## 4013 - 특이한 자석

### 1. 핵심 아이디어
- 4개의 자석이 서로 맞닿는 부분이 다르면 반대로 회전
- 한 자석 회전 시 연쇄적으로 옆 자석 회전 여부 결정
- K번 회전 후 각 자석의 12시 방향(N/S) 합산
- 회전 전 연쇄 방향을 먼저 결정하고 일괄 회전

### 2. 파이썬 코드

```python
import sys
from collections import deque
input = sys.stdin.readline

def rotate(magnet, direction):
    # direction: 1=시계, -1=반시계
    d = deque(magnet)
    if direction == 1:
        d.rotate(1)   # 시계
    else:
        d.rotate(-1)  # 반시계
    return list(d)

T = int(input())
for tc in range(1, T + 1):
    K = int(input())
    magnets = [list(map(int, input().split())) for _ in range(4)]
    # 인덱스: 0=12시, 1=1시, ..., 6=6시(반대), 2=3시, 6=9시

    for _ in range(K):
        num, d = map(int, input().split())
        num -= 1  # 0-indexed

        # 연쇄 회전 방향 결정
        directions = [0] * 4
        directions[num] = d

        # 오른쪽 전파
        for i in range(num, 3):
            # i번 3시 방향(인덱스 2) vs i+1번 9시 방향(인덱스 6)
            if magnets[i][2] != magnets[i+1][6]:
                directions[i+1] = -directions[i]
            else:
                break

        # 왼쪽 전파
        for i in range(num, 0, -1):
            if magnets[i][6] != magnets[i-1][2]:
                directions[i-1] = -directions[i]
            else:
                break

        # 일괄 회전
        for i in range(4):
            if directions[i] != 0:
                magnets[i] = rotate(magnets[i], directions[i])

    # 점수 계산: 12시(인덱스 0) 기준
    score = 0
    for i in range(4):
        if magnets[i][0] == 1:  # S극
            score += (1 << i)

    print(f'#{tc} {score}')
```

## 4014 - 활주로 건설

### 1. 핵심 아이디어
- N×N 지도에서 가로/세로 방향으로 활주로 건설 가능한 수 계산
- 높이 차이가 1인 경우 경사로(길이 X) 설치 가능
- 높이 차이가 2 이상이거나 경사로가 이미 설치된 곳이면 불가
- 가로 행, 세로 열 각각 따로 검사

### 2. 파이썬 코드

```python
import sys
input = sys.stdin.readline

def check(line, X):
    N = len(line)
    used = [False] * N
    i = 1
    while i < N:
        diff = line[i] - line[i-1]
        if diff == 0:
            i += 1
        elif abs(diff) > 1:
            return False
        elif diff == 1:
            # 낮은 쪽(i-1 방향)에 경사로 필요
            start = i - X
            if start < 0:
                return False
            for k in range(start, i):
                if used[k] or line[k] != line[i-1]:
                    return False
                used[k] = True
            i += 1
        else:  # diff == -1
            # 낮은 쪽(i 방향)에 경사로 필요
            end = i + X
            if end > N:
                return False
            for k in range(i, end):
                if used[k] or line[k] != line[i]:
                    return False
                used[k] = True
            i = end
    return True

T = int(input())
for tc in range(1, T + 1):
    N, X = map(int, input().split())
    board = [list(map(int, input().split())) for _ in range(N)]
    ans = 0

    for i in range(N):
        if check(board[i], X):         # 가로
            ans += 1
        if check([board[r][i] for r in range(N)], X):  # 세로
            ans += 1

    print(f'#{tc} {ans}')
```

## 4130 - 사다리 타기

### 1. 핵심 아이디어
- 사다리 가로 연결을 추가해서 특정 출발점에서 목표 도착점에 도달하게 만들기
- 가로줄 추가는 최소 1개
- 위에서 아래로 내려가며 시뮬레이션
- 현재 위치에서 가로줄 따라 이동하는 규칙 구현

### 2. 파이썬 코드

```python
import sys
input = sys.stdin.readline

def go_down(board, N, M, start):
    col = start
    for row in range(N):
        # 왼쪽 가로줄 확인
        if col > 0 and board[row][col-1] == 1:
            col -= 1
        # 오른쪽 가로줄 확인
        elif col < M - 1 and board[row][col] == 1:
            col += 1
    return col

T = int(input())
for tc in range(1, T + 1):
    N, M, X = map(int, input().split())
    board = [list(map(int, input().split())) for _ in range(N)]

    # 이미 조건 만족하는지 확인
    ans = 0
    if go_down(board, N, M, X) == X:
        ans = 0
    else:
        found = False
        # 가로줄 1개 추가 시도
        for r in range(N):
            for c in range(M - 1):
                if board[r][c] == 0:
                    # 인접한 곳에 이미 가로줄 없는지 확인
                    if (c == 0 or board[r][c-1] == 0) and (c+1 >= M-1 or board[r][c+1] == 0):
                        board[r][c] = 1
                        if go_down(board, N, M, X) == X:
                            ans = 1
                            found = True
                        board[r][c] = 0
                        if found:
                            break
            if found:
                break

        if not found:
            ans = 2  # 2개 필요하거나 불가

    print(f'#{tc} {ans}')
```

## 5644 - 무선 충전

### 1. 핵심 아이디어
- A, B 두 사람이 M번 이동하며 배터리 충전
- 매 시간마다 두 사람 위치에서 범위 내 BC 중 최대 충전량 선택
- 두 사람이 같은 BC 선택 가능 → 같은 BC면 절반씩
- 매 시간 완전탐색으로 최적 BC 조합 선택

### 2. 파이썬 코드

```python
import sys
input = sys.stdin.readline

# 방향: 0=정지, 1=상, 2=우, 3=하, 4=좌
da = [0, -1, 0, 1, 0]
db = [0, 0, 1, 0, -1]

T = int(input())
for tc in range(1, T + 1):
    M, A_cnt = map(int, input().split())
    move_a = list(map(int, input().split()))
    move_b = list(map(int, input().split()))
    bcs = []
    for _ in range(A_cnt):
        x, y, c, p = map(int, input().split())
        bcs.append((x, y, c, p))  # 행, 열, 범위, 충전량

    # 위치에서 사용 가능한 BC 인덱스 목록
    def get_bcs(x, y):
        result = []
        for i, (bx, by, c, p) in enumerate(bcs):
            if abs(bx - x) + abs(by - y) <= c:
                result.append(i)
        return result

    ax, ay = 1, 1
    bx, by = 10, 10
    total = 0

    move_a = [0] + move_a
    move_b = [0] + move_b

    for t in range(M + 1):
        ax += da[move_a[t]]
        ay += db[move_a[t]]
        bx += da[move_b[t]]
        by += db[move_b[t]]

        a_list = get_bcs(ax, ay)
        b_list = get_bcs(bx, by)

        best = 0
        for ai in (a_list + [-1]):
            for bi in (b_list + [-1]):
                charge = 0
                if ai != -1:
                    charge += bcs[ai][3]
                if bi != -1:
                    charge += bcs[bi][3]
                if ai != -1 and ai == bi:
                    charge -= bcs[ai][3]  # 같으면 한 번만
                best = max(best, charge)
        total += best

    print(f'#{tc} {total}')
```
