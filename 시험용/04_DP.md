## 1952 - 수영장

### 1. 핵심 아이디어
- 1일권, 1달권, 3달권, 1년권 4가지 이용권으로 최소 비용 계산
- dp[i] = i월까지의 최소 비용
- 각 달마다 4가지 선택 중 최솟값 선택
- 12개월이 고정이므로 DP 크기가 작아 빠름

### 2. 파이썬 코드

```python
import sys
input = sys.stdin.readline

T = int(input())
for tc in range(1, T + 1):
    # 1일권, 1달권, 3달권, 1년권 가격
    p1, p2, p3, p12 = map(int, input().split())
    usage = [0] + list(map(int, input().split()))  # 1~12월 사용 횟수 (1-indexed)

    # dp[i] = i월까지의 최소 비용
    dp = [0] * 13

    for i in range(1, 13):
        # 1일권으로 이번 달 처리
        dp[i] = dp[i-1] + usage[i] * p1
        # 1달권 구매
        dp[i] = min(dp[i], dp[i-1] + p2)
        # 3달권 구매 (i-2, i-1, i 달 커버)
        if i >= 3:
            dp[i] = min(dp[i], dp[i-3] + p3)
        else:
            dp[i] = min(dp[i], p3)  # 처음부터 3달권
        # 1년권은 마지막에 한 번만 비교
        if i == 12:
            dp[i] = min(dp[i], p12)

    print(f'#{tc} {dp[12]}')
```

## 2115 - 벌꿀채취

### 1. 핵심 아이디어
- N×N 지도에서 두 일꾼이 각각 M칸씩 연속 구간을 선택 (같은 행, 겹치지 않게)
- 선택한 구간에서 꿀의 합이 C 이하이면서 제곱합 최대화
- 각 행의 모든 구간(시작점, 길이 M)에 대해 최대 제곱합을 미리 계산
- 두 일꾼이 겹치지 않는 구간 쌍의 최댓값 합산

### 2. 파이썬 코드

```python
import sys
input = sys.stdin.readline

def best_score(row, start, M, C):
    # start부터 M칸 구간에서 꿀 합 C 이하인 경우의 최대 제곱합
    # 부분집합으로 선택
    best = 0
    for mask in range(1 << M):
        total = 0
        sq = 0
        for k in range(M):
            if mask & (1 << k):
                total += row[start + k]
                sq += row[start + k] ** 2
        if total <= C:
            best = max(best, sq)
    return best

T = int(input())
for tc in range(1, T + 1):
    N, M, C = map(int, input().split())
    board = [list(map(int, input().split())) for _ in range(N)]

    ans = 0

    for r in range(N):
        # 각 구간 시작점별 최대 점수 계산
        scores = []
        for start in range(N - M + 1):
            scores.append(best_score(board[r], start, M, C))

        # 두 구간이 겹치지 않는 쌍 탐색
        for s1 in range(N - M + 1):
            for s2 in range(s1 + M, N - M + 1):
                ans = max(ans, scores[s1] + scores[s2])

    print(f'#{tc} {ans}')
```
